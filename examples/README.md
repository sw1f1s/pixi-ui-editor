# Examples

## All Elements Gallery

Open `all-elements.pixiprojectui` in the editor through `Project -> Open Project File`.

The project includes:

- every supported node type: `container`, `graphics`, `componentInstance`, `scrollView`, `list`, `grid`, `mask`;
- every editable node component: `fill`, `texture`, `text`, `shadow`, `outline`, `button`, `slider`, `toggle`, `checkbox`, `radio`, `input`, `textInput`, `dropdown`, `progressBar`, `mask`, `repeater`, `layout`, `scroll`, `scrollView`;
- sample windows, modals, HUD bars, inventory/shop/settings screens;
- texture, atlas, font and data assets;
- reusable component definitions, variants, exposed props, states, tokens, themes and editor panel layout.

Regenerate the bundle after editing the source:

```sh
node examples/all-elements-project.mjs
```
